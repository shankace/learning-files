exists查询
```
SELECT * FROM article WHERE EXISTS (SELECT * FROM user WHERE article.uid = user.uid)
```
EXISTS (subquery) 只返回 TRUE 或 FALSE，因此子查询中的 SELECT * 也可以是 SELECT 1 或其他，官方说法是实际执行时会忽略 SELECT 清单，因此没有区别。
